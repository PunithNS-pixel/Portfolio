## Overview

A personal gaming content platform built to showcase BGMI gameplay highlights, strategies, and community content. The project combines web development skills with gaming passion to create an engaging user experience.

## Key Features

- **Content Management System**: Built a custom CMS for uploading, categorizing, and managing gaming videos and highlights.
- **Responsive Design**: Implemented a mobile-first design approach using HTML, CSS, and Tailwind CSS for optimal viewing across all devices.
- **Video Gallery**: Created an organized video gallery with filtering options by game mode, map, and performance highlights.
- **Community Section**: Developed a blog section for sharing gaming strategies, tips, and community discussions.
- **Performance Analytics**: Integrated analytics to track video performance, viewer engagement, and popular content trends.
- **Social Media Integration**: Connected with YouTube, Instagram, and gaming platforms for seamless content sharing.

## Technologies Used

- **Frontend**: HTML5, CSS3, and Tailwind CSS for modern, responsive design
- **Backend**: Python with Flask for content management and API development
- **Database**: SQLite for lightweight data storage and content management
- **Video Processing**: FFmpeg for video optimization and thumbnail generation
- **Deployment**: GitHub Pages for cost-effective hosting and continuous deployment
- **Analytics**: Google Analytics for tracking user engagement and content performance

## Challenges and Learnings

The main challenge was optimizing video loading times while maintaining quality. This required learning about video compression, lazy loading, and CDN integration. The project taught me how to balance technical implementation with user experience in content-heavy applications.

## Outcome

The platform successfully showcases over 50 gaming highlights and has attracted a growing community of BGMI enthusiasts. The site demonstrates the intersection of technical skills and personal interests, serving as both a portfolio piece and a content platform.
